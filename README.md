# Todo App

A full-stack task management application built using FastAPI and Streamlit. This project was developed independently to gain hands-on experience with REST APIs, frontend-backend communication, CRUD operations, and Python web application development.

## Features

- Create new tasks
- View all existing tasks
- Update entire tasks using PUT requests
- Partially update task fields using PATCH requests
- Delete tasks
- Interactive Streamlit user interface
- FastAPI-powered backend API
- Input validation using Pydantic

## Tech Stack

- FastAPI
- Streamlit
- Pydantic
- Requests
- Docker
- Docker Compose

## Learning Objectives

This project was built to:

- Learn FastAPI fundamentals
- Practice REST API development
- Understand HTTP methods (GET, POST, PUT, PATCH, DELETE)
- Implement data validation with Pydantic
- Build a simple frontend using Streamlit
- Connect frontend and backend services
- Explore Docker-based deployment
